promise基本用法

promise 封装在方法内,比如方法为getPhone，可以进行传参，然后进行一些判断

`
    getPhone(e){
        let p = new Promise(() =>{
            if(e == 0){
                resolve('成功')
            }else{
                reject('失败')
            }
        })
        return p
    },
`

以上的调用方式:,其中resolve回来的参数，在res中，reject中的参数，在err中

`
    this.getPhone(0)
    .then(res =>{
        console.log(res)
    })
    .catch(err =>{
        console.log(err)
    })
`

promise.all，当包含的所有promise中执行完，再进行操作,注:有一个失败就算失败，两个都成功才算成功

`
    let result = Promise.all([this.getPhone(0),this.getPhone(1)])
    result.then(res =>{
        console.log(res)
    })
    .catch(err =>{
        console.log(err)
    })
`
